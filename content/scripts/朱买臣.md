#
朱买臣

根据清传奇《烂柯山》改编改编:沈念禾

# 剧情简介

一个寒儒朱买臣之妻崔氏,好吃懒做,不甘受贫,逼朱买臣写下休书,改嫁张屠户。后朱买臣得中,任会稽太守。
崔氏愧悔交集,想入非非,夜梦冠带。
朱买臣荣归故里,
崔氏拦在马前,欲与其和好。谁知朱买臣当众将一盆水泼出,意为覆水难收,将崔氏羞辱一番。

# 人物
朱买臣、王安道、崔氏、张屠户、王妈妈、差人甲、差人乙、地保、四龙套、四校尉
