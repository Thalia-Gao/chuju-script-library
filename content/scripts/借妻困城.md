# 借妻团城

借妻团城别名:《张古董借妻》整理:易马 杨少华

# 剧情简介

剧情简介张古董助好友李天龙进京赴试,为筹盘费,借己妻充李妻,随李往岳丈家领取亡妻金银首饰。走后,张猛然悔悟,唯恐弄假成真,急忙追赶,至城门口因赌快手玩器纠缠,耽误进城时间,同被关在月城之内。李天龙与张妻翠花亦因岳丈留宿同坐洞房,一夜未宿。张在月城胡思乱想坐卧不安。

# 人物
张古董、古董妻、李天龙、王老大、李玩器、丫鬟
